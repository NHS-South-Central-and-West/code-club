# Intro to Heuristic Evaluation

A Code Club talk given by Jeffrey Morgan-Harrisskitt introducing [Heuristic Evaluation](https://usabilitygeek.com/heuristic-evaluation-introduction/), a user experience method that involves a number of experts evaluating the useability of a product against recognised useability principles.
