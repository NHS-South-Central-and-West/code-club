# Recursive Programming

A Code Club talk given by Jeffrey Morgan-Harrisskitt introducing [recursive programming](https://en.wikipedia.org/wiki/Recursion_(computer_science)), a computational process in which a [function calls itself until some "base condition" is met](https://www.freecodecamp.org/news/recursion-is-not-hard-858a48830d83/). The talk covers what recursion is, what problems it can help solve, and includes some simple examples in R, JavaScript, and Python.
