# Code Club

A repository containing resources from talks given at Code Club sessions, where we discuss various topics related to coding at SCW. The talks are given by anyone that has something that they want to share, and that might help others improve their coding skills. Talks can range from a quick introduction to a certain method to a more in-depth discussion of a particular topic, and the goal is to make coding more accessible to everyone at SCW by providing a platform for people to share their knowledge.

Not every talk will be shared here, as some talks may not include any code or relevant resources that can be shared, and some resources may include sensitive information that cannot be shared publicly. However, where possible, as much of the content/resources from each talk as possible will be shared here.

The resources for each talk are stored in their own subdirectory, with a `README.md` file containing a description of the talk, and any relevant links, and a `resources` subdirectory containing any relevant files. Each talk subdirectory name is prefixed with the date of the talk, in the format `YYYY-MM-DD`, and the name of the talk.
